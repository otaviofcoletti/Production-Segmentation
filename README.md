# Production-Segmentation
Pipeline de segmentação de plantações



Neste repositório temos a seguinte organização:

Na pasta 1 temos a quebra das imagens grandes em blocos para facilitar um posterior processamento por redes neurais

Na pasta 2, temos o módulo de segmentação das imagens e criação do groundtruth de mascaras binarizadas, nesse projeto utilizamos o GLI por conta de alguns testes que fizemos foi o que teve melhor desempenho analisando visualmente
